Criando Repositório no GitHub (www.github.com)

Acessando o GitHub

Acesse a página do GitHub (clique aqui para o GitHub), clique em Sign in ou entrar em português.

Criar Repositório

Após login e senha, clique no ícone de perfil que fica no canto superior direito e depois encontre a opção “Seus Repositórios” (em inglês, Your repositories).

Clique no botão verde “New”, ou em português “novo”.

Instruções
  • No campo Repository name, ou Nome do repositório em português, devemos inserir o nome que desejamos, sem espaços e acentos.
  • Marque também a opção Add a README file (em português, Adicione o arquivo LEIA-ME), pois é uma boa prática e você pode personalizar esse arquivo.
  • No campo Escolha uma licença (em inglês, Choose a license) selecione MIT License, que é uma licença simples, flexível e com isenção de responsabilidades.
  • Por fim, clique no botão verde Create repository (em português, Criar repositório).

Sua tela deve parecer com esta, com o nome que você escolheu.

Parabéns!
